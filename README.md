## 🚀 Overview

This project is a production-ready, client-centric web application built from scratch. It focuses on delivering a seamless user experience across all devices while maintaining strict adherence to Core Web Vitals and advanced SEO standards. 

## 💻 Tech Stack

*   **Framework:** Next.js
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** Swiper.js (for modern, interactive carousels)
*   **Infrastructure:** Serverless Contact Form integration via EmailJS

## ✨ Key Features

*   **100% Responsive Architecture:** Flawless cross-browser and cross-device rendering (Mobile, Tablet, Desktop).
*   **Extreme Performance:** Achieved ~100/100 across Performance, Accessibility, Best Practices, and SEO in Google Lighthouse.
*   **Serverless Communication:** Secure and instant client-to-coach email inquiries without the need for a dedicated backend.
*   **Clean Code Principles:** Component-based architecture with strong typing for scalable and maintainable code.
